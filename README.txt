Press START (Enter) to start the game.
During the game, you can go back to the start screen by pressing SELECT (Back)
UP, DOWN, LEFT, RIGHT buttons will move the helicopter around but the helicopter will not go past the screen and can only go so far to the right of the starting position.
Try to last as long as you without touching any of the green barriers or the ground.
If you die, your final score will be displayed in the screen and you may press SELECT to start over.
